# Módulo 3 – Simulador de Crecimiento del PIB según Adopción de IA

Este módulo incluye un pequeño script en JavaScript que calcula tres escenarios de crecimiento del PIB español para 2026–2030:

- Escenario lento
- Escenario moderado
- Escenario acelerado

El simulador muestra cómo variaría la economía española en función del nivel de adopción de inteligencia artificial.
